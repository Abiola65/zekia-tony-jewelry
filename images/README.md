# Images Guide for Jewelry Website

## Current Images
The website currently uses high-quality jewelry images from Unsplash (a free image service). These are loaded directly from the cloud.

## Adding Your Own Local Images

If you want to add your own images to this folder, follow these steps:

1. **Add image files** to this folder (images/) with names like:
   - ring.jpg
   - necklace.jpg
   - earrings.jpg
   - bracelet.jpg
   - hero.jpg (for background)

2. **Update the HTML** in `index.html` to reference local images:
   - Replace: `src="https://images.unsplash.com/..."`
   - With: `src="images/ring.jpg"`

## Recommended Image Specifications

### Product Images
- **Dimensions**: 400x400px (square format)
- **Format**: JPG or PNG
- **File Size**: 50-200KB (optimized)
- **Background**: White or transparent

### Hero Image (Background)
- **Dimensions**: 1920x1080px minimum
- **Format**: JPG (for smaller file size)
- **File Size**: 200-500KB (optimized)

## Free Image Resources

### Recommended Sites for Jewelry Images:
- **Unsplash**: https://unsplash.com/s/jewelery
- **Pexels**: https://www.pexels.com/search/jewelry/
- **Pixabay**: https://pixabay.com/search/jewelry/
- **Unsplash Jewelry Collection**: Great for luxury and modern jewelry

## Image Optimization Tips

1. **Compress images** to reduce loading time
2. **Use appropriate formats**: JPG for photos, PNG for graphics
3. **Maintain consistent styling** across all product images
4. **Use white or neutral backgrounds** for product photos
5. **Ensure good lighting** for professional appearance
